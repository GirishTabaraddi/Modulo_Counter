Detailed Description

A Modulo N Digit Counter class, that takes the ModuloNCounter as a pointer instance and displays the counter values from 1 to values less then MaxValue of the counter(type). An overflow is called when the current value is more than MaxValue and the current value is set to 0. Once the overflow is caught, the next counter inline starts counting.This is done for as many counters as digits are entered by user.
